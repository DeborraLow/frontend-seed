# `app/`

The `app/` folder is the src root of your web application.

```
app/
  |- assets/                  // put static assets here
  |- components/              // put decoupled components here
  |- data/                    // put seed/mock data here for assemble pages
  |- lib/                     // put internal reusable javascript/sass here
  |- vendor/                  // put 3rd party libraries here
  |- index.hbs                // home page
  |- main.js                  // main application entry point
  |- main.scss                // main application stylesheet
```
